# AI Product Manager

A multi‑agent product planning and analysis app built with Flask, Socket.IO, SQLAlchemy, and CrewAI/Gemini.

- Multi‑agent collaboration (PM, Developer, QA, AI Engineer)
- Real‑time updates over WebSockets
- Project planning, risk assessment, and automation suggestions

## Demo

Run locally:
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python run.py
```
